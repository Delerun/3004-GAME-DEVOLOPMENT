Game Title & Genre : The Real Monster - 2D Platformer, Arcade, Endless Runner
Team Members & Which Level They Designed: Beraat SAĞLAM - I made everything myself
Game Goal: Our main character, a monster, is trying to survive by running away from a human
Core Mechanics: It can run in any direction, jump, and perform wall jumps. I will also add more mechanics later, but since they haven’t been added yet, I didn’t include them here
Controls: The D key moves the character to the right, and the A key moves it to the left. Similarly, the right arrow key moves to the right, and the left arrow key moves to the left. The Space key makes the character jump. The Enter key restarts the game. I will add more key bindings later.
Target Audience & Platform: Everything is for you, my dear teachers, and will be published on Cats and GitHub
